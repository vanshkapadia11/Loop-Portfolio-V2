# 🚀 Loop Portfolio V2

A sleek and modern personal portfolio website built using **React.js**, **Tailwind CSS**, **AOS (Animate On Scroll)**, and **Vite**.  
It features a dark/light mode toggle, animated components, and a fully responsive layout.
